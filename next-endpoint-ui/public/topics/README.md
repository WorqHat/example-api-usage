# Topics Folder

Place topic background images in this folder. Each topic card will use its corresponding image as a background.

## Required Images

- `agents.jpg` - For the "Agents" topic
- `multimodal.jpg` - For the "Multimodal" topic
- `text.jpg` - For the "Text" topic
- `guardrails.jpg` - For the "Guardrails" topic
- `optimization.jpg` - For the "Optimization" topic
- `workflows.jpg` - For the "Workflows" topic

## Supported Formats

- `.jpg`
- `.jpeg`
- `.png`
- `.webp`

## Fallback

If an image is not found, the topic card will automatically fall back to a dark gradient background.

## Image Recommendations

- **Size**: Recommended 800x200px or similar wide aspect ratio
- **Format**: JPG or PNG
- **Style**: Should work well with white text overlay and dark overlay

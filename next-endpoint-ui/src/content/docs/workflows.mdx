# Workflows

Workflows allow you to automate complex processes using WorqHat's orchestration engine.

## Triggering Workflows

Trigger a workflow with JSON data:

```javascript
const response = await fetch('https://api.worqhat.com/flows/trigger/your-flow-id', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${API_KEY}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    email: 'user@example.com',
    data: { /* your workflow data */ }
  })
});
```

## Workflow Types

- **JSON Triggers**: Send structured data to workflows
- **File Uploads**: Upload files to trigger workflows
- **URL Triggers**: Trigger workflows via webhook URLs

## Monitoring Workflows

Track workflow execution using analytics IDs returned from trigger requests.
